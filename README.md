# TextRange

管理 [Range](https://developer.mozilla.org/zh-CN/docs/Web/API/Range) 的工具. 支持导入/导出. 可用于页面批注.

## Installation

```bash
pnpm add text-range
```

## Usage

```ts
// 用户选中文本后, 未选中会抛出异常
new TextRange();
```

## Example

访问 [Live Demo](https://pinghuazhuang.github.io/comments/text-range/) 或者克隆项目, 并执行:

```bash
pnpm install
pnpm dev
```
